# Local CV Event Schema

Required fields: `event_id`, `event_type`, `module`, `source`, `observed_at`, `confidence`, `severity`, and `payload`.

Supported event types: `PACKAGE_DAMAGE_DETECTED`, `PACKAGE_LOADING_MISMATCH`, `PACKAGE_LOADING_VALIDATED`, `LOAD_COMPLIANCE_UPDATED`, `HUB_VISUAL_CONGESTION_CHANGED`.
